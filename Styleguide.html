Website Style Guide

The look that I am trying to achieve with my website is a very simple, minimalistic but also artistic aesthetic. My character, Naledi is very simple but also very artistic.

I tried to show Naledi's artistic side by adding gifs and interesting artsy images for the blog posts while keeping the site very minimal.

Colour palette

initially, the color pallete that I wanted to use is one that is similar to the one illustrated in the image below. After playing around with different coloured backgrounds in my previous iterations, i have decided to stick with a simple gray background, to keep the site looking simple, nice and neat.

The font that I chose to use in this website is Roboto sans-serif; the reason why I chose this font is because I think that it is readable and clean, thus improving the user experience.

Interactivity

Ideally, I would've like my website to be as interactive as the one on the gif above. It looks minimalistic but also very artistic, it looks like it as a perfect balance. It is also very interactive, it has a lot of animations, and it has great feedback.

This is a prototype still and therefore my site will eventualy get to be more interactive and more aesthetically pleasing, in my other iterations.
