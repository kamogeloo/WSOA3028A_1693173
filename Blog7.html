INTERACTIVITY

Interactivity according to Aaron Smuts means when something is responsive, if it gives the user some kind of agency and if it does not respond in a “random fashion” (Smuts p65). Something can also be considered interactive when it requires nontrivial effort, and if it can modify the shape of the text in some way. Hypertext is the organization of information units into connected associations that a user can choose to make, provide the most common form of interactivity when using the Web.

Any kind of user input, this is inclusive of typing commands or clicking the mouse, is a form of input. Inputs in the web/computer outputs forms of interactivity, this can include printouts, sounds, displayed images, text, videos etc.

Interactive web design is one which uses features aimed at creating an environment for a website or web application user to be actively engaged during a visit or use; it can also be used to improve user experience.

Elements of interactivity in the web include:

  • Hyperlinks
  • Graphics/ animation/sound
  • Channels for on-line communication/transaction/feedback
  • Easy navigation/accessibility/organization
  • Speed, Feedback
  • Search Tools
  • Relevance of topics

The benefits of interactive design are that it allows users to customize their user experience through their submission of personal data. It guides users on the site to their desires location through navigation, enables the owners of the websites to expand their reach more through the use of social media and blogs. It improves the overall user experience, and attracts more traffic to the website.
