Semantic markup

Semantic markup is the way in which programmers structure and write their code in HTML. it is HTML that intrioduces meaning to the web page, it is not just what the page looks

the semantic element in HTML include the header, footer, article, section , which are called sementic because of how they specify what the use of the element is.for example, an article is meant to represent enclosen an article.

The semantic elements explain exactly what they are for in the code, making it easier for people to read the code and understand what everything means in the code.

It is very important to use the right tag or element because they each perform thir own function.
